# ShareFare

A community marketplace to share food and reduce waste.

## Features

- **Share Food**: Post extra food items to share with your community.
- **Find Food**: Browse available food items near you.
- **Chat**: Coordinate pickups with secure messaging.
- **Profile**: Track your impact and reputation.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Lucide React Icons
- React Router DOM

## Project Structure

- `src/components`: Reusable UI components
- `src/context`: State management
- `src/pages`: Application views
- `src/types.ts`: TypeScript definitions
- `src/constants.ts`: Mock data and configurations
